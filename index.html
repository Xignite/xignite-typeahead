Getting Started

  1. Get the js and css files from https://github.com/Xignite/xignite-typeahead.

  2. Get the dependencies (jQuery 1.x is recommended if you intend to support IE 6-8):

  3. jQuery

  4. jQueryUI

  5. Create an arbitrary input element in your HTML. The data-xignite-typeahead-key must be specified if a keyParam (in step 4) is defined (required for XigniteSearch):

<input type="text" class="xignite-typeahead" data-xignite-typeahead-key="XigniteGlobalQuotes.GetGlobalDelayedQuote.Identifier" />
  1. Invoke the widget in javascript like so:
<script type="text/javascript">
    $(".xignite-typeahead").xigniteTypeahead({ api: "http://search.xignite.com/Search/Suggest", keyParam: "parameter", q: "term" });
</script>
